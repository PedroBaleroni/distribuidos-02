# distribuidos-02
 Second Class Work. Using Pyro4 to structure a simple example of RMI system


# Responsáveis:
- Pedro Baleroni
- Renan Souto


# INICIALIZAÇÃO

- Rodar o comando:
    
    ```pyro4-ns -n "IP LOCAL"```
    
- Rodar o servidor:
    
    ```py server_crud.py```
    
- Rodar o Programa local Client:
    
    ```py client_crud.py```
    

# INICIALIZATION
- Run:
    
    ```pyro4-ns -n "IP LOCAL"```
    
- Run Server:
    
    ```py server_crud.py```
    
- Run Client:
    
    ```py client_crud.py```
    
    
# RELATÓRIO

Todo o protocolo de Envio é baseado em TCP, no caso de teste, em local.
